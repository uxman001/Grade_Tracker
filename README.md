# Grade_Tracker

A simple Python program that allows users to enter grades for different subjects, display all recorded grades, and calculate the overall average grade.

Features
Input grades for multiple subjects.
Store subject names and grades using a dictionary.
Display all entered grades.
Calculate and show the average grade.
Easy to understand and beginner-friendly.
Technologies Used
Python 3
How to Run
Make sure Python 3 is installed on your system.
Download or clone this repository.
Open a terminal in the project folder.
Run the program using:
python grade_tracker.py

Project Purpose

This project was created to practice basic Python concepts such as loops, dictionaries, user input handling, and arithmetic operations while building a useful grade tracking application.
